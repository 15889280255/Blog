### 基于 vue + node.js 的博客系统

#### 前端(web)
- Vue 2.6.x
- Vue-router
- axios
- vue/cli 4.4.6
- @kangc/v-md-editor 富文本 markdown 预览
- highlight.js 代码高亮

#### 后台管理系统(admin)
- Vue 2.6.x
- Vue-router
- ElementUI
- axios
- vue/cli 4.4.6
- @kangc/v-md-editor 富文本编辑器

#### 后端(server)
- node 
- Express (express: "^5.0.0-alpha.8")
- mongoose
- bcryptjs 加密
- jsonwebtoken
- qiniu 七牛云SDK
